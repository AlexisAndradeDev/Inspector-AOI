Para ejecutar el script regallbrds.py, antes debe existir un archivo regallbrds_dt.di en la ruta C:/Dexill/Inspector/Alpha-Premium/x64/pd/
Este archivo debe contener los datos de registro. La documentacion del archivo regallbrds_dt.di se encuentra en el repositorio de Gitlab del sistema
PyV Alpha Black (https://gitlab.com/Alexismtzan/pyv-alpha-black/) en el archivo python/registro/documentacion para C#/Ejecucion/Estructura_de_datos_de_registro.txt

En el archivo regallbrds_dt.di se escribe la direccion de la carpeta que contiene las imagenes de los tableros desalineados.
En la direccion se deben encontrar las imagenes con el formato de nombre: 
	numero_de_foto.bmp

Ejemplo:
	1.bmp
	2.bmp
	3.bmp
	4.bmp
	5.bmp

Si se va a usar luz ultravioleta, deben existir también imágenes con el formato de nombre:
    numerodefoto-ultraviolet.bmp
Ejemplo:
    1-ultraviolet.bmp
    2-ultraviolet.bmp
    3-ultraviolet.bmp
    4-ultraviolet.bmp
    5-ultraviolet.bmp

Para iniciar el registro, ahora solo hara falta ejecutar el script regallbrds.py y exportara las imagenes resultantes en C:/Dexill/Inspector/Alpha-Premium/x64/pd/
