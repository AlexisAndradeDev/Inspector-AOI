Para ejecutar el script dbg.py, antes debe existir un archivo dbg_dt.di en la ruta /x64/pd/

Este archivo debe contener los datos de debugeo. La documentación del archivo dbg_dt.di se encuentra en este mismo directorio, en el archivo estructura-de-datos.txt

En el archivo dbg_dt.di se escribe la dirección de la carpeta que contiene las imágenes de los tableros alineados con el script regallbrds.py

En la dirección se deben encontrar las imágenes con el formato de nombre: 
	número_de_tablero-luz-board_aligned.bmp

    * Las imágenes de los tableros a debugear deben estar alineadas.

Ejemplo:
	1-white-board_aligned.bmp
	2-white-board_aligned.bmp
	3-white-board_aligned.bmp
	4-white-board_aligned.bmp
	5-white-board_aligned.bmp
	
Si se usa luz ultravioleta, las imágenes llevan el nombre de:
    número_de_tablero-ultraviolet-board_aligned.bmp
    
Ejemplo:
    1-ultraviolet-board_aligned.bmp
    2-ultraviolet-board_aligned.bmp
    3-ultraviolet-board_aligned.bmp
    4-ultraviolet-board_aligned.bmp
    5-ultraviolet-board_aligned.bmp

Para iniciar el debugeo, ahora sólo hará falta ejecutar el script regallbrds.py y exportará las imágenes resultantes.
